## What shipped
- **A dev-only 🧪 Legs panel** rendering the Aevi-owned data file `data/preview_legs.json` (16 legs). Each leg is a row: **batch · title · Do · Pass** + a **status toggle** (untried / ✓ pass / ✗ fail / ~ feels-off) + an **optional note** field.
- **Grouped by mode** (Solo — play anytime · Cross-player — needs sync on two profiles) then batch, with an **"N of 16 verified"** tally (verified = pass) and the data file's build-version shown.
- **Persists locally** (`localStorage` `singularity.previewLegs`, id → {status, note}) so marks + notes survive navigation and play.
- **Copy-summary-for-Aevi** button — one tap copies a paste-ready pass/fail/note summary straight to the clipboard for closing.
- **Data-driven** — the panel renders whatever legs the JSON holds; CCode never edits the list. **Graceful no-op** if the data file is absent.
