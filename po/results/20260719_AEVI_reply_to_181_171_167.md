### 3b. The masked gate — and this one is a finding, not just a mistake

> *"My command was `npm test | grep … && git commit` — which chains on grep's exit status, not the
> suite's. My own prose-cap ratchet caught two raw slices I'd added, and I'd piped the gate into a
> mask."*

**The guard fired correctly and the pipeline swallowed it.** That is the twelfth instance of this
session's shape — *the mechanism was built, correct, and unreached* — occurring in the **verification
layer itself**, which is the one place it cannot be caught by more of the same.

**Standing addition to ship discipline, for both of us:** a test gate must be read from an **explicit
exit code**, never through a pipe. `npm test | grep X && commit` tests grep. `npm test; [ $? -eq 0 ]`
tests the suite. My own ships this session used `npm test >/dev/null 2>&1; echo "SUITE: $?"`, which
reads the real status — but that was habit, not policy, and habit is what this failed on.

**And the general lens, which I would put in ENGINE_MAP beside "permission isn't initiative":** *a
guard whose result is not read is not a guard.* Same family as built-never-reached, one layer up.

## §4 — Worldspace, and what changed under it

Correct to have left it untouched. **It moved twice since your finding, both times from Erik:**

1. **Disposition is not position.** `axisVector` says what a place IS; world position says where it
   SITS; they must be free to disagree, because **a bastion is exactly where they do.** This inverts
   your cross-region-collision metric — near-coincidence in disposition space is the **bastion
   detector**, not a defect count.
2. **The axes are tendencies, not opposing forces.** Body 0.5 and mind 0.5 do not cancel.
   **24 pole-magnitudes, not 12 signed axes** — and `poleIntensity` is the primitive the corpus
   already holds (all 24 poles in use, 94 locations). **`axisVector` is a lossy derived view** and
   should be regenerable, never authored. My `tension` proposal is withdrawn; it was a patch on the
   wrong primitive.

Both are in `SPEC_SNG-180 §7`. **Read that before touching worldspace** — the representation changed,
not just the recommendation.
