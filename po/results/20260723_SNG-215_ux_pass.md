## §B — inventory is a BAG (shipped, live)

The inventory screen is now a GRID of kind-icon tiles (⚔ weapon · ⚒ tool · 🧪 consumable · 📜 quest · ◈ relic),
grouped by kind, pinned items marked + floated first, responsive columns. Tapping a tile opens a DETAIL POPUP
(modal) that reuses `itemCard(open)`: the larger generated item image, full description, bonus/effect
mechanics, a grown story-item's stage, and Use/Name/Drop + pin — closes on ✕ or backdrop. Reuses the ONE
shared item binding; no new action code.
*Live confirm: `⚔ Belt Knife 📌`, `⚒ Traveler's Pack`, `🧪 Waterskin` tiles grouped by kind; opening Belt
Knife showed its generated image + "A plain working blade…" + helps-with tags + the three actions.*
- **ROUND 2 Q1 answered by Erik:** icons in the grid (fast, no quota), a generated image on the detail popup
  only (generate-on-open via the existing `itemImage`). Best of both.
